# DataHandleWithPandas

A simple Python project demonstrating basic data handling operations using the pandas library.

## Overview

This project shows fundamental pandas operations including:
- Creating DataFrames from dictionaries
- Statistical analysis with `describe()`
- Checking for missing values with `isnull()`

## Files

- `data.py` - Main script containing pandas data manipulation examples

## Requirements

- Python 3.x
- pandas

## Installation

```bash
pip install pandas
```

## Usage

Run the main script to see basic data analysis:

```bash
python data.py
```

## Sample Output

The script will display:
- Statistical summary of the dataset (count, mean, std, min, max, quartiles)
- Count of null values in each column

## Example Data

The project uses a simple dataset with:
- Age: [25, 30, 35]
- Salary: [50000, 60000, 80000]
